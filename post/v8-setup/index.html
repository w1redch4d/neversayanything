V8 Setup

Getting the v8 source

For this blog we will be focusing mostly on the setup in linux, for macOS and windows you can find the documentation on the v8’s offical website

considering that you have git install , first let us setup depot_tools using the following commands:

 git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git

Add depot_tools to the front of your PATH (you will probably want to put this in your ~/.bashrc or ~/.zshrc). Assuming you cloned depot_tools to /path/to/depot_tools:

 export PATH=/path/to/depot_tools:$PATH

Now, get the V8 source code, including all branches and dependencies:

mkdir /path/to/v8/
cd /path/to/v8/
fetch v8
cd v8

to synchronize and update the v8 dependencies use:

gclient sync

Building v8 from source

Considering you have one of the debian distributions, run the following command after switching to the v8 directory to install dependencies related to v8

./build/install-build-deps.sh

now compile the source as per your need:

tools/dev/gm.py x64.release

or

tools/dev/gm.py x64.debug
